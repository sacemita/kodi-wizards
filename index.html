Bienvenidos a Kodi-wizards

. Estoy cogièndole la mano al código [Markdown](https://guides.github.com/features/mastering-markdown/ using GitHub Flavored Markdown, espero que este trabajo lo pueda publicar bajo la licencia de Creative Commons by 3.0 - Unported. El cometido de este proyecto es juntar una serie de configuraciones predeterminadas para arrancar el kodi enseguida de instalarlo, con el conjunto màs relevante de addons que haya en el mercado al dìa de la fecha. Esto para que el que lo usa no pase un año buscando addons que funcionen, asimismo como enbellecer la configuraciòn del kodi con fondos de pantallas fanart de pelis conocidas y que son distribuios tambièn gratuitamente.
